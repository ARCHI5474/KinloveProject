# KinloveProject

## プロジェクト概要
KinloveProject（キンラブ・プロジェクト）は、Webサービス開発を主軸に、SNS支援・創作・コミュニティ運営など多角的に活動するクリエイティブプロジェクトです。

---

## 想い（公式サイトより）
KinloveProjectが掲げる想い：

1. **チャレンジ**  
   クリエイターが自分で考え、実践する。知識に経験を加えて最高のものを生み出す。

2. **創る情熱が喜びに変わる**  
   情熱をもって作られたものは、利用するユーザーにとっての喜びになる。

3. **フロー体験**  
   あることに没頭し、それを極めることで知識と経験が糧になる。

4. **上の上を目指す**  
   「できた」で満足せず、さらなる高みを目指す。

5. **ウェルビーイング**  
   身体的・精神的健康と自己実現を両立できるプロジェクトを目指す。

**Keep pushing the limits.**  
*- 限界を超え続けよう -*

---

## プロジェクトについて（公式Noteより）
KinloveProjectは、「創作活動」を通じて社会にある様々な壁をなくし、  
誰もが自分らしく生き生きとした毎日を歩めるように、という想いから始まりました。

### 活動の柱
- **新しいWebサービスを作る**  
  「ヒマロリンク」「プロディ」「LikeStudy」など、直感的で使いやすいWebサービスを開発・運営。

- **クリエイターの仲間づくりを応援する**  
  クリエイターコミュニティ「HimaRoCreators」を運営し、交流・協力・成長を支援。

### 運営方針
- クリエイター・ユーザーの声を最優先
- 新技術・トレンドを積極的に導入
- 透明性を重視し、問題は早期共有・解決
- AI活用時は必ず複数回チェックを実施

---

## 主な活動
- Webサービス開発（KindAssistant / HimaRoLink など）
- SNS支援・コミュニティ運営（HimaRoCreators）
- クリエイター支援

---

## プロダクト紹介

### HimaRoLink（コードネーム: LongSquareLink）
**シンプルさと軽量さを重視した超軽量リンク集サービス**

- 低スペック端末でも快適に動作
- 迷わず目的のサイトへアクセス可能
- フィードバックを重視した改善サイクル

---

### KindAssistant（コードネーム: HimaRoLinkChoice）
**機能性・直感性・シンプルさを両立したWebサービス**

- ブラウザのトップページ向けリンク集
- 直感的でわかりやすいUI
- 最新技術・トレンドを継続的に導入
