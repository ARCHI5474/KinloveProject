# KinloveProject

## プロジェクト概要
KinloveProject（キンラブ・プロジェクト）は、Webサービス開発を主軸に、SNS支援・創作・コミュニティ運営など多角的に活動するクリエイティブプロジェクトです。

### 公式サイトより
> KinloveProjectは、Webサービス開発を主軸に、SNS支援、創作、コミュニティ運営など多角的に活動するクリエイティブプロジェクトです。  
> KinloveProjectが掲げる想い：
> 1. **チャレンジ**  
>    クリエイターが自分で考え、実践する。知識に経験を加えて最高のものを生み出す。
>
> 2. **創る情熱が喜びに変わる**  
>    クリエイターが情熱をもって作ったものは、利用するユーザーにとって喜びになる。
>
> 3. **フロー体験**  
>    あることに没頭し、それを極めることで知識と経験が糧になる。
>
> 4. **上の上を目指す**  
>    「できた」で満足せず、さらなる高みを目指すことで、プロジェクトやクリエイターの人生に良い影響を与える。
>
> 5. **ウェルビーイング**  
>    身体的健康・精神的健康と自己実現を両立できるプロジェクトを目指す。
>
> Keep pushing the limits.  
> -限界を超え続けよう-

### 公式Noteより
> Kinlove Projectは、「創作活動」を通じて、社会にある様々な壁をなくし、誰もが自分らしく、生き生きとした毎日を歩めるようにという願いから始まりました。  
> 私たちの活動の原動力は「創る情熱は、使う喜びになる。」というモットーで、インクルーシブな社会の実現を目指しています。
>
> 活動の柱：
> 1. **新しいWebサービスを作る**  
>    「ヒマロリンク」「プロディ」「LikeStudy」などのWebサービスを開発・運営。誰もが直感的に利用できる、シンプルで使いやすい機能を提供。
>
> 2. **クリエイターの仲間づくりを応援する**  
>    SNSで活動している方々が集まるコミュニティ「HimaRoCreators」を運営。仲間同士の交流・協力・知名度向上をサポート。(KinloveProjectで創作活動を行うクリエイターが主に参加)
>
> 活動運営のポイント：
> - クリエイターやユーザーの声を最優先に反映  
> - 新しい技術やトレンドを積極的に学習・導入  
> - プロジェクトの透明性を確保し、問題は早期に共有・解決
>
> AIを活用した広報活動も行うが、誤情報や不適切な情報が発信されないよう、必ず複数回チェックしています。
>
> Kinlove Projectは、元々「暇人が作ったサイト」(2021年10月開設)から始まり、コミュニティ化や名称変更を経て、2025年11月1日に正式設立されました。

### 主な活動
- Webサービス開発（KindAssistant/HimaRoLinkなど）
- SNS支援・コミュニティ運営（HimaRoCreators）
- クリエイター支援・プロジェクト改善
